## ğŸ”‘ Key System Components

### 1. **Flexible Input Handler**
- **Accepts any CSV format** from any supplier
- **Auto-detects columns** (ITEM# vs SKU# vs Product_ID)
- **Cross-references** product database with DataDive competitor data
- **Normalizes data** into internal schema

### 2. **Self-Learning Knowledge Base**
- **5 JSON/MD files** storing institutional knowledge
- **Auto-loads before each analysis** (system educates itself)
- **Updates after each analysis** (continuous learning)
- **Extracts from meeting notes** (learns from team discussions)

### 3. **Pattern Recognition Engine**
- **6+ success patterns** identified from past analyses
- **2+ rejection patterns** for auto-filtering
- **Pattern matching** during analysis (+15 points if match)
- **New pattern discovery** as more products analyzed

### 4. **Multi-Factor Scoring Model**
- **8 weighted factors** (Margin 25%, Competition 20%, etc.)
- **0-100 score calculation** with confidence level
- **Decision thresholds** (85+ = STRONG GO, 70-84 = GO, etc.)
- **Self-calibrating weights** adjust based on accuracy

### 5. **Table-Based Primary Output**
- **Master comparison table** for portfolio-level view
- **Rank-ordered products** by score and viability
- **At-a-glance decisions** (GO/NO-GO with key metrics)
- **Learning insights** showing model improvements

### 6. **Feedback Loop System**
- **Logs predictions** for future validation
- **Tracks actual decisions** from Todd
- **Compares predicted vs actual** outcomes
- **Adjusts weights automatically** after 3-5 validations
