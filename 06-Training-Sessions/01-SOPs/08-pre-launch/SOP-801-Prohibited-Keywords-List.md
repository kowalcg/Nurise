# SOP-801: Prohibited Keywords List

**Nurise Listing Optimization**

---

## Purpose
This reference document helps Amazon sellers avoid using restricted keywords that can lead to lower rankings, listing suppression, or account suspension. Regularly review this list when creating or updating product listings.

## When to Use
- Writing new product titles, bullet points, or descriptions
- Creating A+ Content
- Updating backend search terms
- Reviewing listings for compliance
- Before submitting any listing for approval

## Prerequisites
- [ ] Access to your Amazon listing content
- [ ] Understanding of your product category restrictions

---

| Category | Prohibited Keywords |

### 6. Guarantee & Quality Claims
**Unsubstantiated claims not allowed:**

| Prohibited | Safe Alternative |
|------------|------------------|
| Guarantee / Guaranteed | "Satisfaction policy included" |
| Proven | "Designed to..." |
| Certified | Only if actually certified |
| Approved | Only if actually approved |
| High quality | Describe specific qualities instead |
| Tested | Only if third-party tested with proof |
| Buy with confidence | Remove entirely |
| Unlike other brands | Remove comparison |
| Validated | Only with proof |
| Recommended by | Only with documented endorsement |

### 8. Enhanced Brand Content (A+ Content) Restrictions

**Never include in A+ Content:**

| Restriction | Explanation |
|-------------|-------------|
| Blurry or low-quality images | Minimum resolution requirements |
| Competitor products or seller authorization | No comparative content |
| Copyright, trademark, or registered symbols | Use sparingly, only if owned |
| Customer reviews | Cannot include review content |
| Editorial or third-party quotes | Must be independently verifiable |
| Grammatical or punctuation errors | Content quality requirement |
| Logos from other brands/organizations | Only your own branding |
| Company as seller or distributor | Focus on brand, not seller |
| Pricing or promotion information | No price mentions |
| Product used for criminal activity | Obvious prohibition |
| Time-sensitive information | No dates, limited offers |
| Web links or redirecting language | No external URLs |

---

### 9. Product Title Requirements

**Titles MUST:**
- Contain product-identifying information (e.g., "hiking boots", "umbrella")
- Be under 200 characters (including spaces)

**Titles MUST NOT contain:**
- Decorative characters: `~ ! * $ ? _ ~ { } # < > | * ; ^ ¬ ¦`
- Gift phrases: "perfect gift", "great present"
- Promotional phrases: "free shipping", "100% quality guaranteed", "best price"
- All caps words (except brand names)

## Compliance Procedure

### Step 1: Pre-Listing Review
Before publishing any listing:

1. **Scan title** for prohibited keywords
2. **Review bullet points** one by one
3. **Check description** for claims
4. **Verify backend search terms** don't include banned keywords
5. **Review A+ Content** for prohibited elements

### Step 2: Use Search Function
```
In your word processor:
1. Press Ctrl+F (Windows) or Cmd+F (Mac)
2. Search for each category of banned keywords
3. Replace or remove any matches
```

### Step 3: Document Changes
- Keep a log of removed/replaced keywords
- Note why changes were made
- Update listing notes in DataDive

## Safe Keyword Alternatives

| Instead of... | Use... |
|---------------|--------|
| "Cure" | "Supports", "Promotes" |
| "Guaranteed" | "Designed for reliability" |
| "Best" | "Premium", "Professional-grade" |
| "Organic" | "Natural ingredients" (if unverified) |
| "Antibacterial" | "Easy to clean" |
| "Non-toxic" | "Safe materials" |
| "Eco-friendly" | "Thoughtfully designed" |
| "Weight loss" | "Supports active lifestyle" |

## Common Mistakes

| Mistake | Consequence | Prevention |
|---------|-------------|------------|
| Using "organic" without certification | Listing suppression | Only use if USDA certified |
| Health claims in supplements | Account warning | Use structure-function claims only |
| Competitor names in keywords | Trademark violation | Never reference competitors |
| "Best" or "guaranteed" in title | Listing flagged | Use descriptive alternatives |
