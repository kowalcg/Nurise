# DataDive Analysis

This directory contains all DataDive analysis exports and screenshots for product validation.

## Directory Structure

```
DataDive-Analysis/
├── Master-Keyword-Lists/ (Excel exports of MKL for each product)
├── Product-Scorecards/ (Screenshots/exports of scorecard results)
├── Roots-Analysis/ (Screenshots of roots analysis)
├── Competitor-Analysis/ (Competitor analysis summaries)
└── Deep-Dive-Reports/ (Deep dive reports for priority products)
```

## DataDive Analysis Checklist

For each product, complete:

1. **Niche Creation & Setup**
   - [ ] Create niche in DataDive
   - [ ] Set marketplace to Amazon Canada
   - [ ] Add ASINs to niche
   - [ ] Run initial dive

2. **Master Keyword List (MKL)**
   - [ ] Export MKL to Excel
   - [ ] Document total search volume
   - [ ] Count relevant keywords (>30 required)
   - [ ] Identify top keywords
   - [ ] Analyze competitor rankings

3. **Roots Analysis**
   - [ ] Screenshot roots page
   - [ ] Identify main keyword roots
   - [ ] Verify >3 main roots requirement
   - [ ] Document search volume distribution

4. **Competitor Analysis**
   - [ ] Identify top 10 competitors
   - [ ] Document competitor strength ratings (Weak/Strong/Very Strong)
   - [ ] Record ratings, reviews, sales estimates
   - [ ] Analyze market concentration
   - [ ] Check for Amazon Retail presence

5. **Product Scorecard**
   - [ ] Run scorecard analysis
   - [ ] Screenshot scorecard results
   - [ ] Document risk scores
   - [ ] Complete all 6 factors

6. **Deep Dive** (for priority products)
   - [ ] Run deep dive analysis
   - [ ] Export comprehensive competitor landscape
   - [ ] Document market opportunity assessment

## File Naming Convention

- MKL: `MKL-{ITEM#}-{DESCRIPTION}.xlsx`
- Scorecard: `Scorecard-{ITEM#}-{DESCRIPTION}.png`
- Roots: `Roots-{ITEM#}-{DESCRIPTION}.png`
- Competitor: `Competitors-{ITEM#}-{DESCRIPTION}.md`
